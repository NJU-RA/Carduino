  @ 文件夹介绍
    该文件夹主要存放制作循迹小车需要用到的软件

  * Arduino ide
    这是编写小车代码需要用到的集成开发环境。Arduino的编程语言是一种类C语言，如果有C语言的基
    础，那真是 couldn't be better! 当然，要是没有C语言基础，也不要紧，万事开头难嘛！

    如果你是用的是windows系统，请在arduino-1.8.4-windows 文件夹(整体),打开文件夹中的
    arduino.exe可直接使用。

    如果你是用的是 Mac OS X系统，可以选择arduino-1.8.4-macosx 文件夹(整体)。

    当然，最新版的Arduino ide可以在官网上下载。
    传送门:https://www.arduino.cc/en/Main/Software?setlang=cn

  * Arduino 库函数索引
    这个文件夹中包含Arduino中一些基本库函数的使用方法，如果有C语言编程基础就可知道，这其实就
    是一些封装好的函数，调用相关函数即可实现相应功能。

  * 蓝牙模块
    其中HC05指令集是用来调试蓝牙的指令。
    单片机调试助手是用来进行串口通信调试的，当然也可以用来进行蓝牙调试。
